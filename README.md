# 💻 Algoritmos e Lógica de Programação | FATEC

Este repositório armazena meus estudos, exercícios e projetos práticos desenvolvidos durante o curso de **Tecnologia em Informática para Negócios**. O foco aqui é a resolução de problemas lógicos, estruturação de dados e conceitos de sistemas.

---

## 🚀 Sobre o Repositório

O objetivo deste espaço é documentar minha evolução técnica, com foco em:
*   **Lógica de Programação:** Desenvolvimento de algoritmos estruturados e resolução de problemas.
*   **Arquitetura de Sistemas:** Estudos sobre o funcionamento interno de computadores e sistemas operacionais.
*   **Sistemas Operacionais:** Conceitos de threads, processos e concorrência.

---

## 🛠️ Tecnologias e Ferramentas

*   **VisualG:** Desenvolvimento de algoritmos em pseudocódigo (Portugol).
*   **Linguagens de Programação:** Implementações em linguagens como C ou Java (conforme o progresso acadêmico).
*   **Git & GitHub:** Versionamento e organização de código.
